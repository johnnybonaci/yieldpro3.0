# YIELDPRO

YieldPro is an system automation and optimization that allows agencies and online marketers to automate 99% of their online marketing optimization process.

## Installation

Clone Repository

```bash
git clone git@github.com:beatsmedia/yieldpro.git
```

## Installation with Docker

```bash

cd yieldpro
chmod +x install.sh
sh install.sh


```

## Installation without Docker

```bash

cd yieldpro
chmod +x reset.sh
sh reset.sh

```

## Host

```bash

#Virtual Host
http://localhost:3000


```
